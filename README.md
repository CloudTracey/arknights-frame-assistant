## 下载及启用方法

1. 从网页右边的 [Release](https://github.com/CloudTracey/arknights-frame-assistant/releases) 下载 exe 文件
2. 与游戏一同启动（没有先后顺序要求）
3. 对所希望修改的按键和设置进行自定义
4. 点击保存设置或应用设置，即可在游戏中生效

<br>

## 功能特性

### 1. 过帧与精细操作

| 功能 | 说明 |
|------|------|
| 一键技能 | 一键开启鼠标所指单位的技能 |
| 一键撤退 | 一键撤退鼠标所指单位 |
| 暂停选中 | 在暂停时选中鼠标所指的场上单位 |
| 暂停技能 | 在暂停时开启鼠标所指的场上单位的技能 |
| 暂停撤退 | 在暂停时撤退鼠标所指的场上单位 |
| 前进 33ms | 前进游戏 1 倍速下的 1 逻辑帧（游戏为 1 秒 30 逻辑帧） |
| 前进 166ms | 前进游戏 0.2 倍速下的 1 逻辑帧（选中单位或者待部署区时的 1 逻辑帧） |

> **注意**：由于目前 PC 端的输入限制，所有的**暂停中操作**均为**巨缝操作**，使用体验远不如模拟器的按键方案，具体原因见 [部分功能的又一次失效通知](https://www.bilibili.com/opus/1167682130438782995)

### 2. 全按键自定义

支持自定义绑定的功能包括：

| 功能 | 默认按键 | 功能 | 默认按键 |
|------|----------|------|----------|
| 按下暂停 | F | 前进 33ms | R |
| 松开暂停 | Space | 前进 166ms | T |
| 切换倍速 | D | 一键技能 | E |
| 暂停选中 | W | 一键撤退 | Q |
| 干员技能 | S | 暂停技能 | 鼠标前侧键 |
| 干员撤退 | A | 暂停撤退 | 鼠标后侧键 |
| 模拟左键点击 | Z | | |

- 支持绑定键盘上的绝大多数按键
- 支持绑定除鼠标左键外的鼠标按键（包括滚轮和侧键）
- 使用 `ESC` 或 `BACKSPACE` 可清除绑定

### 3. 其他功能

- **自动更新**：AFA支持自动更新
- **随游戏自动启动**：可配置随小助手启动时自动运行游戏
- **进程监控**：游戏关闭后可自动退出小助手（避免触发其他游戏的反作弊系统）

## 特殊设置

### 仅对明日方舟进程生效

本程序的按键功能仅对明日方舟进程生效，不会影响游戏以外的操作

### 随游戏进程关闭自动退出

本程序基于 AutoHotkey 开发，由于大部分 MMORPG 或者竞技类网游的反作弊系统对使用 AutoHotKey 开发的程序非常敏感，很可能将本程序当做作弊程序或非法程序处理，导致游戏无法正常启动，甚至导致使用者的账号遭受封禁，因此强烈建议开启小助手的"随游戏进程关闭自动退出"功能，避免因忘关工具触发反作弊

<br>

## 注意事项

### 游戏内的按键设置需为默认设置，请按照以下说明修改：

- 战斗内变速：D
- 释放技能：E
- 撤退干员：Q

### 关于游戏内帧数

如果你的屏幕刷新率**大于等于**120 赫兹，那么可以开启垂直同步，且小助手内的"游戏内帧数"可以设置成 120
<br>

如果你的屏幕刷新率**低于**120 赫兹，那么不要开启垂直同步，且确保小助手内的"游戏内帧数"设置与游戏里设置的帧数一致

### 如果不小心关闭了设置界面

从电脑右下角托盘区可以找到小助手的图标，右键后点击**打开设置**即可重新打开设置界面

<br>

## 常见问题

### Q: 程序启动后没有反应？

A: 请检查以下几点：
1. 确认程序以管理员权限运行
2. 确认游戏已经启动
3. 检查右下角托盘区是否有小助手图标，包括隐藏的托盘区图标（那个小箭头）

### Q: 按键绑定后不生效？

A: 请检查：
1. 游戏内按键设置是否为默认设置
2. 按键是否与其他软件冲突
3. 是否在游戏窗口内操作

### Q: 如何关闭自动更新检查？

A: 在设置界面的"其他设置"标签页中，取消勾选"自动检查更新"即可

### Q: GitHub Token 是什么？需要配置吗？

A: GitHub Token 用于提高 API 请求配额。只有在提示 API 配额超限时才需要配置，一般情况下无需使用

### Q: 如何彻底退出小助手？

A: 退出方式：
1. 右键点击托盘区图标，选择"退出"即可
2. 如果启用了**随游戏进程关闭自动退出**，游戏退出后，AFA也会随之一同退出

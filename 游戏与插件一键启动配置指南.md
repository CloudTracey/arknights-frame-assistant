### 🎮 游戏与插件一键启动配置指南

为了解决启动游戏时多次弹出管理员确认窗口的问题，请按照以下三个步骤操作。我们将创建一个自动脚本，只需确认一次即可启动所有插件和游戏。

#### 第一步：制作启动脚本

1. 在电脑上新建一个文本文档（记事本）。
2. 将下面的代码复制进去。
3. **关键：** 请将代码中的路径修改为你电脑上实际的软件路径。

```batch
@echo off
:: ------------------------------------------------
:: 自动启动脚本
:: ------------------------------------------------

:: 1. 启动第一个插件 (无需等待)
start "" "D:\Tools\Plugin1.exe"

:: 3. 等待 2 秒让插件准备好
timeout /t 2 /nobreak >nul

:: 4. 启动游戏主程序 (带参数)
:: 注意：请确保 -launcher_lang 等参数写在引号外面
start "" "D:\Games\MyGame\Game.exe" -launcher_lang=zh-cn -launcher_sub_channel=1

:: 退出黑框
exit
```

1. 保存文件时，选择“**所有文件**”，将文件名命名为 `Arknights.bat` (注意后缀必须是 **.bat**)。

##### 举例如下：

```batch
@echo off
:: ------------------------------------------------
:: 自动启动脚本
:: ------------------------------------------------

:: 1. 启动增强插件 (无需等待；使用 start 命令确保脚本不等待其结束就继续执行)
:: 这里的 "" 是为了防止路径中有空格导致 start 命令解析错误，必须保留
start "" "D:\Games\Android\AFA\AFA-v1.0.8.exe"

:: 3. 等待 2 秒让插件准备好
timeout /t 2 /nobreak >nul

:: 4. 启动游戏主程序 (带参数)
:: 注意：这里同样保留第一个空引号 ""；同时请确保 -launcher_lang 等参数写在引号外面
start "" "D:\Games\Android\Hypergryph Launcher\games\Arknights Game\Arknights.exe" -launcher_lang=zh-cn -launcher_sub_channel=1

:: 退出黑框 (脚本窗口)
exit
```



#### 第二步：创建桌面快捷方式

1. 找到刚才做好的 `Arknights.bat` 文件。
2. 鼠标**右键**点击它，选择 **发送到** -> **桌面快捷方式**。
3. （可选）你可以右键点击桌面上新生成的快捷方式，选择“重命名”，把它改成“启动游戏”。

#### 第三步：开启管理员权限 (核心步骤)

这一步将合并所有的安全弹窗，只需确认一次。

1. **右键**点击桌面上这个新的快捷方式，选择 **属性 (Properties)**。
2. 点击右下角的 **高级 (Advanced)** 按钮。
3. 勾选 **以管理员身份运行 (Run as administrator)**。
4. 连续点击两次 **确定** 保存。

------

### ✅ 验证效果

现在，请双击桌面的这个快捷方式：

1. 系统会弹出**一次**确认窗口（UAC），请点击 **“是”**。
2. 随后，两个插件和游戏将自动依次启动，**不会**再出现任何弹窗干扰。

------

**管理员注 (Admin Note):**

- 如果遇到“找不到文件”的错误，请检查脚本中的路径是否保留了双引号。
- 此方法利用了 Windows 的权限继承机制，是微软推荐的安全做法。

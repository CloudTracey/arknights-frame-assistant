# 核心功能测试

如果涉及大量修改，请在提交PR前执行下方的测试项

## 测试前配置验证

- [ ] 游戏内按键设置为默认（战斗变速=D，释放技能=E，撤退干员=Q）
- [ ] 小助手内游戏内帧数与游戏设置一致
- [ ] 低刷新率屏幕未开启垂直同步

## 暂停与帧操作

- [ ] 额外暂停键A（按下触发）能正确暂停游戏
- [ ] 额外暂停键B（松开触发）能正确暂停游戏
- [ ] 切换倍速能正确切换游戏速度（发送 f+g）
- [ ] 前进33ms能正确过1逻辑帧
- [ ] 前进166ms能正确过0.2倍速下的1逻辑帧
- [ ] 暂停选中能在暂停时选中鼠标所指单位

## 操作功能

- [ ] 干员技能能正常触发
- [ ] 干员撤退能正常触发
- [ ] 一键技能能点击并触发技能
- [ ] 一键撤退能点击并撤退单位
- [ ] 暂停技能能在暂停时触发技能
- [ ] 暂停撤退能在暂停时撤退单位

## 设置管理测试

- [ ] 设置能正确保存到配置文件
- [ ] 重启后能加载上次保存的设置
- [ ] 重置默认设置能恢复初始值
- [ ] 应用设置后立即生效（不关闭窗口）
- [ ] 取消设置后不保存修改

## 热键绑定测试

- [ ] 点击输入框能进入按键录制模式
- [ ] 按键录制后能正确显示按键名称
- [ ] 按ESC或Backspace能清除按键
- [ ] 按键冲突检测能正确提示
- [ ] 修改按键后能正确应用新绑定

## 游戏监控测试

- [ ] 开启"自动退出"后，游戏关闭时程序自动退出
- [ ] 未开启"自动退出"时，游戏关闭程序继续运行
- [ ] 程序仅在Arknights.exe激活时响应热键

## GUI界面测试

- [ ] 窗口标题显示正确版本号
- [ ] 托盘右键菜单能打开设置
- [ ] 游戏帧数下拉选项（30/60/120）能正确选择
- [ ] 自动打开设置窗口选项生效
